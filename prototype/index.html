1. 개요

이 문서는 Nova Skin의 디자인 컨셉을 보여주기 위한 예시 문서입니다. Nova Skin은 buma 스킨의 모던하고 정갈한 레이아웃과, Liberty 스킨의 정보 밀도 높은 사이드바 및 커스터마이징 기능을 결합했습니다.

알림. 이런 형태의 알림 박스는 buma의 bNotification 컴포넌트 스타일을 따릅니다.

2. 상세

각 스킨에서 어떤 요소를 가져왔는지 정리하면 다음과 같습니다.

2.1. buma에서 가져온 요소

  • 페이지 상태(정상/없음/오류)에 따라 색이 바뀌는 히어로 헤더
  • 둥근 모서리의 카드형 알림(notification) 박스
  • 우측 하단 플로팅 이동 버튼
  • data-theme 속성 기반 다크 모드 전환
  • 모바일 전용 검색 바 분리 패턴

2.2. liberty에서 가져온 요소

  • 우측 고정 사이드바의 실시간 "최근 변경" 피드
  • 도구 드롭다운의 풍부한 관리 메뉴 구성
  • 문서 헤더 아래 탭형 도구(편집/역사/토론/역링크) 배치
  • CSS 커스텀 프로퍼티 기반 브랜드 컬러 시스템
  • 문서 하단의 리비전 정보 + 저작권 표시 푸터

4. 여담
